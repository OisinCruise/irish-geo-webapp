About Irish Historical Sites GIS

Our Mission

The Irish Historical Sites GIS project aims to make Ireland's rich cultural heritage accessible to everyone through modern web mapping technology. By combining accurate geographic data with detailed historical information, we provide an engaging platform for exploring castles, monasteries, ancient monuments, and archaeological sites across the island.

Features

  • Interactive map with multiple basemap options (street, satellite, terrain)
  • Bilingual support in English and Irish (Gaeilge)
  • Dark and light theme modes
  • Geolocation to find sites near you
  • Drawing and measurement tools
  • Filtering by site type, county, and historical era
  • Coordinate display and search
  • Responsive design for all devices

Technology Stack

Backend Django 4.2 LTS
Database PostgreSQL + PostGIS
API Django REST Framework
Mapping Leaflet.js + OpenStreetMap
Spatial Data GeoJSON / WGS84
Frontend HTML5 / CSS3 / ES6

Data Sources

Our historical site data is compiled from various authoritative sources including the National Monuments Service, Ordnance Survey Ireland, and academic research. County and province boundaries are derived from official OSi datasets.

API Access

Developers can access our data through a RESTful API that returns GeoJSON-formatted responses. The API supports filtering, pagination, and spatial queries.
